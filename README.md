## Manually ##
- extract this
- place the folder JoystickDirection in documents/Arduino/libraries/
- start your ide and enjoy!

## with arduino ide ##
- extract this
- compress the folder JoystickDirection
- Sketch -> Include Library -> Add.ZIP Library and JoystickDirection.zip
